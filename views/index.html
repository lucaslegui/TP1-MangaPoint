API RESTful

Bienvenido a la API sobre información de Mangas y Anime

Esta API permite interactuar con una base de datos realizada con MONGODB de mangas y adpataciones a peliculas y series. A continuación, encontrarás los endpoints disponibles y cómo usarlos.

Endpoins

Mangas

  • 🔒GET localhost:3000/mangas - Devuelve todos los mangas.
  • GET localhost:3000/mangas/title - Devulve todos los mangas ordenados alfabéticamente por titulo. por su ID.
  • GET localhost:3000mangas/title/:title - Devuelve mangas filtrados por titulo.
  • GET localhost:3000/mangas/genre/:genre - Devuelve mangas filtrados por género.
  • 🔒POST localhost:3000/mangas - Añade un nuevo manga al catálogo.
  • PUT localhost:3000/mangas/title/:title - Actualiza un manga existente seleccionado previamente por el titulo, se puede cambiar todo su contenido.
  • 🔒DELETE localhost:3000/mangas/:id - Elimina un manga específico.

Anime

  • 🔒GET localhost:3000/adaptations - Devuelve peliculas y series.
  • GET localhost:3000/adaptations/:id - devuelve el anime por ID.
  • 🔒POST localhost:3000/adaptations - Añade un nuevo objeto al catalogo (requiere como minimo un titulo, tipo (serie o película) y descripcion).
  • PUT localhost:3000/adaptations/:id - Actualiza un anime existente seleccionado previamente por el ID, se puede cambiar todo su contenido.
  • 🔒DELETE localhost:3000/adaptations/:id - Elimina un manga específico (solo se desactiva).
