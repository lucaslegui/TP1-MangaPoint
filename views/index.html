Bienvenido a la API de Mangas

Esta API permite interactuar con una base de datos de mangas. A continuación, encontrarás los endpoints disponibles y cómo usarlos.

Endpoins

  • GET /mangas - Devuelve todos los mangas.
  • GET /mangas/:id - Devuelve un manga específico por su ID.
  • GET /mangas/genre/:genre - Devuelve mangas filtrados por género.
  • POST /mangas - Añade un nuevo manga al catálogo.
  • PUT /mangas/:id - Actualiza un manga existente.
  • PATCH /mangas/:id - Actualiza un manga existente sin modificar los datos existentes en este.
  • DELETE /mangas/:id - Elimina un manga específico.
