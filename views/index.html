API RESTful

Bienvenido a la API de Mangas

Esta API permite interactuar con una base de datos de mangas. A continuación, encontrarás los endpoints disponibles y cómo usarlos.

Endpoins

  • GET localhost:3000/mangas - Devuelve todos los mangas.
  • GET localhost:3000/mangas/:id - Devuelve un manga específico por su ID.
  • POST localhost:3000/mangas - Añade un nuevo manga al catálogo.
  • PUT localhost:3000/mangas/:id - Actualiza un manga existente.
  • PATCH localhost:3000/mangas/:id - Actualiza un manga sin modificar los demás datos existentes en este.
  • DELETE localhost:3000/mangas/:id - Elimina un manga específico.
  • GET localhost:3000/mangas/genre/:genre - Devuelve mangas filtrados por género.

Mangas por orden

  • GET localhost:3000/title - Devuelve mangas ordenados por título, en orden ascendente.
  • GET localhost:3000/title/:title - Devuelve mangas filtrados por titulo.
  • GET localhost:3000/genre/:genre - Devuelve mangas filtrados por género.
